# 脳卒中PT文献要約システム

毎朝8時にPubMedから脳卒中理学療法の最新エビデンスを自動収集・要約します。

## 特徴
- **PubMed E-utilities API** で実際の論文データを取得（ハルシネーション防止）
- 過去5年以内の **RCT・SR・MA** を優先
- 各論文に **PICO** 付き
- 引用元の **PubMedリンク・DOIリンク** 付き
- 毎日5分で読める要約フォーマット
- 24テーマを日替わりローテーション

## 閲覧方法
- GitHubアプリ（iOS）でこのリポジトリを開く
- SafariでGitHub.comを開いてブラウズ
